# Agent Spec

## Overview
Build an agent that evaluates credit risk for Romanian companies.
The agent should retrieve company data from the ANAF MCP server and use it to support a risk assessment response.

## Example Use Cases
- User asks for credit risk of a Romanian company by name or identifier.
- Agent fetches relevant company data from ANAF MCP server and summarizes risk factors.
- Agent returns a structured risk assessment with key supporting signals.

## Tools Required
- ANAF MCP server: https://demoanaf.ro/mcp
  - Purpose: Retrieve Romanian company data needed for risk analysis.
  - Auth: Not specified yet (assume public/demo endpoint until clarified).

## Constraints & Safety Rules
- No explicit safety constraints provided by user.
- Agent should avoid fabricating unavailable ANAF data and state uncertainty when data is missing.

## Success Criteria
- Agent can accept a company query and return a clear credit-risk evaluation.
- Agent uses ANAF data as evidence in its response.
- Agent output includes concise rationale for risk level.
